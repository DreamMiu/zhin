知音 (or Zhin)

一个基于 icqq 协议的 QQ 机器人开发框架

轻量、优雅、热更、拥抱 icqq

快速开始
查看 API
GitHub

轻量

精简内部功能，仅内置系统级的常用插件和适配器，其他功能均通过插件来实现

优雅

知音的内部实现尽可能符合大众开发思维，无论是阅读源码，还是开发插件，都能事半功倍

热更

知音内置热更插件，让开发者在开发时避免频繁重启进程，从而降低账号风险概率

拥抱 icqq

知音内置了 icqq 适配器（Zhin 作者维护的 oicq 分支），快速对接 QQ 平台
